## 2. Metabase - Open-Source BI Alternative

**URL:** https://www.metabase.com/

**Why it's interesting:**
- Top-rated open-source BI tool (consistently recommended over Power BI for self-hosting)
- No-code query builder with SQL support
- Embeddable dashboards
- Slack/email scheduling built-in
- Works with PostgreSQL, MySQL, SQL Server (common NHS DBs)
- Free and open-source vs Power BI's licensing costs

**For John's NHS workflow:**
- Could replace Power BI for certain use cases
- Email automated reports directly from dashboards
- Self-host on NHS infrastructure (data stays local)

## 3. Apache Superset - Enterprise-Grade Open Source BI

**URL:** https://superset.apache.org/

**Why it's interesting:**
- Airbnb's BI platform now Apache top-level project
- Visual SQL query builder
- Extensive visualization options
- SQLAlchemy connector support (works with most databases)
- Row-level security (important for NHS patient data)
- Used by NASA, LinkedIn, Dropbox

**For John's NHS workflow:**
- Strong security features suitable for healthcare
- Can connect to NHS SQL Server/PostgreSQL
- Rich dashboard capabilities
