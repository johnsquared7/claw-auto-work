## üêç Python Data Tools: Beyond Pandas

### Ibis - The Pandas Alternative with Multiple Backends
**Underrated gem** from the creator of pandas.
- Similar API to pandas but supports multiple backends
- Backends include: pandas, DuckDB, SQL servers (including SQL Server - NHS relevant!)
- Write once, switch execution backend
- **NHS fit:** Could connect directly to NHS SQL Server instances

### Polars Native Plotting (NEW)
Polars recently added `.plot` directly to DataFrame namespace.
- Built on hvPlot for interactivity
- No need to convert to pandas just for charting
- Further reason to migrate from Pandas

### DuckDB Extensions
DuckDB now has rich extension ecosystem:
- **httpfs** - Read data directly from URLs/S3
- **spatial** - Geospatial analysis
- **arrow** - Zero-copy Arrow integration
- **pg_catalog** - PostgreSQL compatibility

## üìä SQL Server Tuning Tools

### Built-in Tools (Free, Already Available)
- **Query Store** - Built into SQL Server 2016+, tracks query performance history
- **Dynamic Management Views (DMVs)** - System views for performance diagnostics
- **Extended Events** - Lightweight tracing (replaces deprecated Profiler)
- **Execution Plans** - Visual analysis of query performance

### dbForge Studio for SQL Server
- SQL Query Profiler for tuning
- Visual query builder
- Index analysis recommendations
- **Note:** Commercial tool but has free edition

### Brent Ozar's Free Resources
- sqlforema.com (free setup)
- Community-driven best practices
- Wait stats analysis
