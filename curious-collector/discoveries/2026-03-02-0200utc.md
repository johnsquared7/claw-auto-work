## ü¶Ü DuckDB + Polars: The Modern Analyst Stack

**Why it's interesting:** For NHS analysts drowning in Excel/csv files, DuckDB offers something revolutionary: SQL queries directly on files without loading them into a database. Combined with Polars (Rust-powered, multi-threaded DataFrames), you get a duo that's 10-100x faster than pandas for large datasets.

**Key insight:** DuckDB processes only necessary columns, executes in cache-friendly vectors, and has a query optimizer‚Äîsomething pandas lacks entirely.

- **DuckDB**: `SELECT * FROM read_csv_auto('patients.csv')` ‚Äî query CSVs with SQL
- **Polars**: pandas-compatible API but with native multi-threading

**Use case for NHS:** Large patient datasets, waiting list analytics, cross-referencing multiple CSVs without Excel memory limits.
