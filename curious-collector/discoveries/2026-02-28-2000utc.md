## 1. Polars - Blazing Fast DataFrame Library

**URL:** https://github.com/pola-rs/polars

**Why it's interesting:**
- Written in Rust, runs 10-100x faster than pandas in many benchmarks
- Zero required dependencies, 70ms import time (vs 520ms for pandas)
- Supports streaming mode for datasets larger than RAM
- Lazy + eager execution with automatic query optimization
- Multi-threaded, SIMD-optimized
- Can handle 250GB+ datasets on a laptop
- Python API is pandas-like but more expressive

**For John's NHS workflow:**
- Could replace pandas for large NHS dataset processing
- Works with Apache Arrow (columnar format)
- Great for automating recurring report data transformations
